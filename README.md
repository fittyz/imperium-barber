# Imperium Barber — Site Premium (protótipo)

Pequeno site estático demonstrando um design premium para a barbearia "Imperium Barber".

Arquivos principais:

- `index.html` — estrutura e conteúdo
- `css/styles.css` — estilos, animações e responsividade
- `js/script.js` — pré-carregador, navegação e pequenos scripts

Como usar:

1. Abra `index.html` no navegador (ou use Live Server no VS Code).
2. Substitua os links do WhatsApp e as imagens pela sua mídia real.
3. Para publicar, envie os arquivos para qualquer host estático (Netlify, Vercel, S3, etc.).

Personalizações recomendadas:

- Atualizar número do WhatsApp em `index.html` (busque por `5511999999999`).
- Trocar o mapa do Google para o endereço real (iframe em `index.html`).
- Substituir imagens da galeria por fotos profissionais da barbearia.
- As imagens de placeholder estão salvas localmente em `assets/images/`.
